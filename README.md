# Web Scraper, Quotes

A minimal Python web scraping example that retrieves quotes and authors from the public demo website `https://quotes.toscrape.com/`. This project demonstrates how to perform HTTP requests, parse HTML, and extract relevant data using widely adopted Python libraries.

## Features

- Sends a request to a target URL
- Parses HTML content using BeautifulSoup
- Extracts all quotes and corresponding authors
- Outputs results to the terminal in a numbered format
- Uses a publicly available demo site intended for scraping practice

## Technologies Used

- Python 3
- requests
- beautifulsoup4 (BeautifulSoup)

## Installation

This project requires Python 3, along with two external libraries.

Install both via pip:

```bash
pip install requests beautifulsoup4
